It works!

This default welcome page is based on the Ubuntu page, which in turn was based on the original Debian page supplied in the original packaging. It is used to test the correct operation of the Apache2 server after installation. If you can read this page, it means that the Apache HTTP server installed at this site is working properly. This file (located at /var/www/html/instructions.html should no longer be displayed once an index.[html|php|py|pl] exists in the /var/www/html/ directory.

Please ensure that you remove or replace this file (located at /var/www/html/) before continuing to operate your HTTP server.

Default Configuration

This docker image is based on the php-apache2 image, with minor changes to support Python, PHP and Perl scripts. It is not meant for production use, more of a simple repeatable base dockerised environment for hosting scripts whilst being developed..

Document Roots

The document root by default is /var/www/html with scripts also beind available in /var/www/cgi-bin. Please ensure you understand the risks, and act accordingly before exposing files and information using this server image.

The default Ubuntu document root is /var/www/html. You can make your own virtual hosts under /var/www. This is different to previous releases which provides better security out of the box. This docker image supports the Ubuntu approach fully to ensure it is as useful, flexible and secure as possible.
